## 📌 Featured Projects

### 🕐 Time Series Forecasting & Anomaly Detection
**Toolkit and app for production-grade time-series analysis**

A comprehensive suite for time-series analysis, forecasting, and anomaly detection — from dataset processing to model evaluation and visual exploration.

- **Tech Stack**: Python, Pandas, Scikit-learn, Prophet, TensorFlow, Streamlit
- **Key Features**:
  - 📓 Reproducible notebooks demonstrating preprocessing, feature engineering, baseline forecasting
  - ⚙️ Time-series cross-validation and backtesting workflows
  - 📊 Interactive dashboard for exploring forecasts and anomalies
  - 🔄 Production-ready pipelines with error handling
- **Get Started**:
  ```bash
  git clone https://github.com/billysambasi/Time_Series_Forecasting.git
  cd Time_Series_Forecasting
  python -m pip install -r requirements.txt
  ```
